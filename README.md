## AI-Optimized Smart Grids

# About the Project

This project simulates a smart grid system using Artificial Intelligence to improve energy distribution. It integrates renewable energy sources (solar and wind) and uses machine learning models to predict electricity demand. The system efficiently manages energy by prioritizing renewable sources, storing excess energy in batteries, and using the grid only when necessary. It also provides visual insights through an interactive dashboard.

# How to Run the Project
# 1. Clone the repository
- git clone https://github.com/your-username/AI-Optimized-Smart-Grids.git
- cd AI-Optimized-Smart-Grids

# 2. Install dependencies
- pip install -r requirements.txt

# 3. Run the application
- streamlit run app.py
